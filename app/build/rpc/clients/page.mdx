# Client Libraries

Celestia Node API can be accessed through various client libraries in different programming languages.

## Official Clients

### Golang Client
- **Repository**: [celestiaorg/celestia-openrpc](https://github.com/celestiaorg/celestia-openrpc)
- **Tutorial**: [Golang Client Tutorial](https://docs.celestia.org/developers/golang-client-tutorial)
- Official Go client library for interacting with Celestia Node API

### Golang Tutorial
Learn how to build applications using the Golang client with our comprehensive tutorial.

## Community Clients

### Rust Client
- **Repository**: [eigerco/celestia-node-rs](https://github.com/eigerco/celestia-node-rs)
- **Tutorial**: [Rust Client Tutorial](https://docs.celestia.org/developers/rust-client-tutorial)
- Community-maintained Rust implementation

### Python Client
- **Repository**: [grumpyp/celestia-node-client-py](https://github.com/grumpyp/celestia-node-client-py)
- Community-maintained Python client for Celestia Node API

### TypeScript Client
- **Repository**: [ashishbhintade/cntsc](https://github.com/ashishbhintade/cntsc)
- Community-maintained TypeScript/JavaScript client

## Getting Started

Each client library provides methods to interact with the Celestia Node RPC API. Choose the language that best fits your project requirements.

For API reference and method documentation, see the [Node API](/build/rpc/node-api) section.
